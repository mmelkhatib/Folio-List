# Folio-List
A Demo Angular 2 App connected to a MongoDB instance

# Requirements 

1. Node 7 and over - https://nodejs.org/en/. 
2. NodeMon - npm install -g nodemon 

# Set up 
1. Clone into the directory of your choice 
2. npm install
3. cd client
4. npm install
5. cd .. (back to the root directory)
6. run nodemon
